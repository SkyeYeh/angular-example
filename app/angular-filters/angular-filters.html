AngularJS Filters

AngularJS 過濾器

  • currency - 格式化數字為貨幣格式。
  • date - 格式化日期為特殊格式。
  • filter - 從數組項中選擇一個子集。
  • json - 格式化物件為 JSON 字串。
  • limitTo - Limits an array/string, into a specified number of elements/characters.
  • lowercase - 格式化字符串為小寫。
  • number - 格式化數字為字串。
  • orderBy - 根據某個表達式排列數組。
  • uppercase - 格式化字符串為大寫。

表達式中添加過濾器

The name is {{ lastName | uppercase }}

The name is {{ lastName | lowercase }}

向指令添加過濾器

Looping with objects:

  • {{ name.name + ', ' + name.country }}

currency 過濾器

Price: {{ price | currency }}
